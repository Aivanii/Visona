Скраб

Скраб — косметическое средство для лица или тела, которое содержит специальные частицы для очищения и отшелушивания мертвых клеток. Активные вещества любого скраба — абразивные частицы на основе эмульсии, глины, крема, геля. В качестве абразивных частиц выступают косточки ягод и фруктов, очищенные сахар, соль, песок, растения.

Существует несколько видов скраба: ‌
Глиняные: подойдут для сужения пор, успешно убирают жирный блеск. Можно использовать для комбинированного и жирного типа кожи. ‌
Кремовые: это мягкий состав для чувствительной и слишком сухой кожи. ‌
Гелевые: подходят всем и являются универсальными.

Основные правила применения косметики с абразивными частицами для очищения просты:
Подбирать состав, подходящий для конкретного участка кожи, например, нельзя использовать средство для ног на лице; применять скраб желательно вечером, поскольку во время сна улучшается процесс деления и восстановления клеток; лицо лучше промыть предварительно теплой водой, чтобы расширились поры.

Внимание! Сама техника использования скраба простая: нужно нанести на влажную кожу небольшое количество средства, легкими движениями распределить по обрабатываемому участку, избегая наиболее чувствительных зон, 3 минуты помассировать, затем смыть теплой водой. После обработки кожи очищающим средством можно нанести питательный крем. Тогда в свободные поры проще проникнут активные вещества, а эффект будет более заметным.
